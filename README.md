# Amir Kapetanovic — Digital Business Card

A single-page, mobile-first digital CV / contact card.

**Live site:** https://arsenevrnd.github.io/CV_Amir/

## Files

- `index.html` — the page (self-contained: CSS + JS inline)
- `og-image.png` — social share preview image (1200×630)
- `CV_Amir_Kapetanovic_2026.pdf` — downloadable CV
- `.github/workflows/deploy-pages.yml` — auto-deploys to GitHub Pages on push to `main`

## How it deploys

On every push to `main`, the GitHub Actions workflow publishes the site to
GitHub Pages. Make sure Pages is enabled with **Settings → Pages → Build and
deployment → Source: GitHub Actions**.
